# GisDetection
Sistema proactivo y automatizado para la detección temprana de cambios en el uso y cobertura del suelo (ej. loteamientos y deforestación) en el Parque Nacional Tunari (PNT), Cochabamba, Bolivia. Emplea Redes Neuronales Convolucionales (CNN) sobre imágenes Sentinel-2 y validación con Aerofotogrametría (VANTs).
